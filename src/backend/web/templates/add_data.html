Add Data to TBA!

Do you have some data you'd like to add to The Blue Alliance? This page will provide you with the resources you need to help contribute to TBA.

Adding Match Videos

If you have links to YouTube recordings of match video, you can link them on TBA. You can suggest videos individually from the detail page for a given match, and if you have a YouTube playlist for an event, click the "Add Match Videos!" button on the "Event" page and input the playlist link. If you're webcasting and archiving an event and want to directly add videos as they get uploaded, you can request write keys.

Event Webcasts

If you have the link to the webcast for an event, you can suggest it from the event's detail page, or from The Blue Alliance homepage if the event is currently active. We make an effort to support many stream types, but Twitch.tv, Livestream, and YouTube are most common.

Team Media

You can suggest various types of team media to TBA from a team's detail page in a specific year. You can suggest robot pictures, reveal videos, and robot CAD. Team social media accounts, like Twitter, Facebook, Instagram, GitHub, and GitLab are not year-specific, and persist from year to year.

Adding a New Offseason Event

Do you know of an offseason event missing from TBA? You can suggest an event to be added here. The request will be reviewed and added shortly.

Adding Offseason Event Results

FMS Sync

Offseason events have the ability to have their match results, team list, and other data synchronized to FIRST Headquarters infrastructure using the same mechanisms that are used in-season. If your offseason-event is using FMS Sync, it is automatically added to TBA. To register your offseason with FIRST, refer to the instructions on frc-events.

Match videos, livestreams, and awards would still need to be added using the mechanisms listed above though.

Offseason FMS sync only supports standard playoff formats, so if your event does something unique, you can still sync Qual results, but you will need to use the "Offline" methods to import playoffs.

Offline FMS

If your offseason event does not have FMS Sync back to Headquarters, or uses a special playoff format, that's not an issue either.

The recommended strategy is to use the "FMS Companion" app running on the FTA Toughbook. See the details here . This app can sync data, including full score breakdowns and unique playoff formats, to TBA if internet is available, or collect a full archive that you can later import via Event Wizard after the fact. You will need to obtain write keys before the event.

Alternately, export FMS Reports (in excel format) before you tear down your field. You can manually import them on the Event Wizard after the event. This method will not include rich score breakdowns of match results, however.

Cheesy Areana

Cheesy Arena is an alternative field management system. It supports direct sync to TBA if you obtain write keys before the event.

FMS Reports

We can also import FMS Report exports into The Blue Alliance. You can either obtain write keys and use the Event Wizard to input the reports, or post them in our Facebook Group.

Other

For other data you'd like to add, reach out on our Facebook Group or drop us a line, we'll be happy to help!
